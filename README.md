# NovaSplit 💸

A personal expense splitting app built with React + TypeScript. Track daily lends and split trip expenses with friends.

## Features

- **Daily Lending** — track who owes you and who you owe
- **Trips** — split group expenses for trips or events
- **Settle Up** — automatically calculates who pays whom
- **Members** — manage your friends list

## Tech Stack

- React 18
- TypeScript
- Vite
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```
